Comprehensive AutoCAD Commands Reference

  • L - Line: Draw straight lines. Example: L → specify start/end points.
  • C - Circle: Center and radius. Example: C → center point → radius.
  • REC - Rectangle: Opposite corners. Example: REC → first corner → opposite.
  • TR - Trim: Cut objects. Example: TR → select cutting edges → trim.
  • EX - Extend: Lengthen to boundary. Example: EX → boundary → extend object.
  • CO - Copy: Duplicate objects. Example: CO → base point → destination.
  • MI - Mirror: Flip objects. Example: MI → select → mirror line.
  • AR - Array: Pattern copies. Example: AR → select → rows/columns.
  • PL - Polyline: Connected lines/arcs. Example: PL → multiple points.
  • DIM - Dimension: Add measurements. Example: DIM → linear/angular.
  • A - Arc: Curved lines. Example: A → start → end → radius.
  • O - Offset: Parallel copies. Example: O → distance → select object.
  • F - Fillet: Rounded corners. Example: F → radius → select edges.
  • CHA - Chamfer: Beveled edges. Example: CHA → distances → select.
